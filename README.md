# Модуль "Ideas" для проекта "ideas-service-4":

## Краткое описание:
Основной микросервис, в котором хранятся идеи, их статусы и авторы;

## Основные задачи:
- Управление Идеями

## Инструкция по запуску проекта:
```
./mvnw spring-boot:run
```

## Предварительный стек:
- SpringBoot v3.3.4
- Apache Kafka
- Camunda
